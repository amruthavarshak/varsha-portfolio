
# Varsha Kurella — Developer Portfolio (React + Vite + TS)

Production-grade personal site built with React, Vite, TypeScript, Tailwind. JSON-driven content, a11y-friendly, PWA, and deploy-ready.

## New Pages
- `/resume` — Resume viewer with **Download PDF** (client-side via jsPDF)
- `/hire` — Recruiter-friendly highlights with quick contact
- `/settings` — Theme presets (rebrand palettes)

## Getting Started
```bash
npm install
npm run dev
```

## Environment
Create `.env` (or copy `.env.example`):
```env
VITE_CONTACT_EMAIL=varsha.kurella@gmail.com
VITE_ANALYTICS_ID=
```

## Build
```bash
npm run build && npm run preview
```
